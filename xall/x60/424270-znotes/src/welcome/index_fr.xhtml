ZNotes 0.9.24
Programme pour créer et gérer des ntoes.
Enregistrer tout ce que vous aimez. De simples notes textes et des pages téléchargées sur Internet, ainsi que des fichiers multimédia. Stockez vos fichiers et les contacts de votre carnet d'adresses dans des notes. Rangez vos notes dans des blocs-notes, organiser les hiérarchiquement dans des catégories et en utilisant des étiquettes de couleur. Affichez et modifiez le code source de vos notes.

Fonctionnalités
  • Blocs-notes
    • Les notes sont stockées dans des blocs-notes. Chaque bloc-notes a une hiérarchie de catégories et dispose de son propre jeu d'étiquettes de couleurs.
  • Système hiérarchique des catégories
    • Les notes sont regroupées en catégories qui forment une structure hiérarchique. Dans chaque catégorie, l'utilisateur peut créer un nombre illimité de sous-catégories et n'importe quel nombre de notes. Les notes sont faciles à faire passer d'une catégorie à l'autre. Vous également définir l'ordre des sous-catégories et l'ordre des notes dans chaque catégorie, vous pouvez tout réorganiser à chaque instant. Tout ceci se fait d'un simple clic de souris.
  • Étiquettes multicolores
    • À chaque note peut être attribué un certain nombre d'étiquettes définies par leur nom et leur couleur. Les notes peuvent être filtrées par une étiquette spécifique. En plus de la structure hiérarchique des catégories cela donne de nombreuses possibilités de gestion de vos notes.
  • Pièces-jointes: fichiers et contacts
    • Un nombre quelconque de fichiers peut être attaché à une note. Le programme ne sauve pas les chemins de fichiers, mais des copies des fichiers dans la note. En outre, vous pouvez lier vos notes avec des contacts du carnet d'adresses. Un simple clic permet de visualiser et d'éditer la fiche de contact.
  • Importer et sauvegarder localement des copies de pages Web
    • Lors de l'importation d'une page Web le programme enregistre votre copie locale personnelle de cette page avec toutes les ressources connexes, tels que des fichiers images, les styles et les scripts.
  • Editer le code source
    • L'accès au code source de la note peut étendre de manière significative les possibilités d'édition. Après avoir modifié le code source basculez tout simplement vers l'onglet « Design » pour voir le résultat des dernières modifications dans le code source.
  • Consulter et éditer dans un onglet séparé
    • Ouvrir des notes dans des onglets séparés vous permet de basculer rapidement entre les notes et simplifie le traitement des information.
  • Imprimer
    • Les utilisateurs peuvent imprimer non seulement la note, mais aussi son code source. L'aperçu avant impression permet de configurer les différents paramètres de mise en page de la note.
  • Synchroniser et stocker les informations dans le Cloud
    • Il est facile d'utiliser des services de synchronisation populaires tels que Dropbox qui permettent le stockage de vos données dans le Cloud. La synchronisation est effectuée instantanément, car les blocs-notes et les notes sont stockés dans le système de fichiers sous forme d'une arborescence de répertoires et de petits fichiers individuels.
  • Méthodes alternatives pour stocker les informations
    • Il est possible de mettre en œuvre et permettre à un pilote personnalisé d'accéder aux informations. Par exemple, il est possible de mettre en œuvre le stockage de données non dans le système de fichiers, mais dans une base de données ou en utilisant des Web services.
