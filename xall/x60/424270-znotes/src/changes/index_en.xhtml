17.08.2014
ZNotes 0.9.23
Fixed bugs and added new features
  • Added support for remote debugging
  • Fixed infinite loop in the browser on some pages
  • Fixed scroll behavior(bounce) in the editor
  • Fixed bug interfering with other add-ons
  • Web-clipper saves notes with a type application/xhtml+xml, by default
  • The open/save note dialog stores the original type and name of the notes
  • Fixed saving messages containing inline images
  • Fixed many bugs and did many improvements in web-clipper:
    • added support for CSS at-rules
    • added support for XML namespaces
    • added mode of replacement tags in HTML5
    • added mode of saving only the active CSS rules
    • syntax errors in the saved documents are corrected on the fly

Fixed bugs and added new features
  • Added support for multiple selections in the editor's source mode
  • Added the ability to change the type of an existing note
