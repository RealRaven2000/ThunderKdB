"Максимизируйте полезное пространство экрана, используя скрытие заголовка окна Firefox"

ПРОЧТИ:

   Пожалуйста, взгляните в начале на диалог параметров. Там есть описание основных функций.

Важно:

   Отчеты об ошибках/проблемах весьма приветствуются на сайте поддержки. Пожалуйста, не размещайте их на сайте дополнений Mozilla.

Краткие примечания:

  • (Информация обновлена для Firefox 4+)
    Перетаскивание     окна:  теперь работает с AeroSnap/unSnap в Win.Vista/7. (возможно, но не обязательно, с небольшими косяками)    :
    Вы можете перемещать 'не максимизированное' окно за настраваемый заголовок или использовать элементы "панелей инструментов":
    "Заголовок страницы", "Тянуть окно", "Интервал" или "Растягивающийся интервал" (заполнители панелей).
    Обратите внимание, "Тянуть окно" отображается только в НЕ максимизированном окне со СКРЫТЫМ заголовком. (т.е. по необходимости).
  • Темы / скины: Работает и выглядит хорошо с ChromiFox основной темой, с ChromiFox Extreme и Cfx Extreme CARBON. Также с Strata40 (+StrataBuddy) темой.
    - (Совет: Попробуйте поиграть с де/активацией пользовательского заголовка для каждой темы. Где то выглядит лучше с новой опцией маленького заголовка, где то без заголовка).
    - Также хорошо смотрится с легкими темами. Можете посмотреть скрины на сайте.
    - Для Chromifox, Strata40 и др. тем: Существуют дополнительные скины кнопок Мин,Макс,Закрыть, помимо обычных.
    - С помощью "ПКМ по панели » Изменить", кнопки Мин,Макс,Закрыть можно поместить на любую панель. А в настроеках скрыть стандартные кнопки управления окном.
  • Меню с доступом к настройкам дополнения доступно в 'Домашней' кнопке Firefox, меню инструментов, Мин,Макс,Закрыть кнопках по ПКМ,
    элементе управления "Тянуть окно" и т.д.
