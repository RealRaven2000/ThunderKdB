Hide Caption Titlebar Plus     [ Support Site ]   [ Main Page ]  

"Maximize your screen content space by controlling the presence of Firefox window's Title bar (caption)"

README:

   Please give a look at the Options Dialog first! There are descriptions of the main features.

Important:

   Bug/Issue reports are very welcomed in the Support site. Please don't post them in the Addons Mozilla site.

Brief notes:

  • (Info updated for Firefox 4+)
    Dragging/moving     window:  now with AeroSnap/unSnap in Win.Vista/7. (some controls still has some unwanted unSnapping)    :
    You can drag the 'un-maximized' window using the custom caption (titlebar), and also you can use the following items in the "Customize Toolbar" window: the "Web-Page-Title" objects, the "Drag Window" controls, the "Space" or "Flexible Space" (fill easily toolbars' empty spaces).
    Please note that these "Drag Window" controls only shows in FF when it's NOT maximized and the standard caption is hidden. (when they are really needed).
  • Themes / Skins: Works and looks good with ChromiFox basic theme, with ChromiFox Extreme and Cfx Extreme CARBON. Also with Strata40 (+StrataBuddy) theme.
    - (Tip: Try playing with de/activating the cc (custom-caption) with each theme, with Cfx-basic I think it looks better with new small-cc option. Others themes go well without cc).
    - Also looks very good with Mozilla's Persona skins. (see preview in homepage)
    - For Chromifox, Strata40 and others themes: There is an extra option for Min,Max,Close buttons skins following the theme's own buttons skins.
    - Via "Customize Toolbar" window, the Min,Max,Close buttons can be placed in any toolbar. (Also, in options dialog, you can hide the buttons located at topright corner)
  • There is a Context-Menu (right-click) accesible from the Main Firefox Application Button, System (Min,Max,Close) buttons and "Drag-Window" control, for opening Options dialog, etc.
